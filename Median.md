In statistics, Median is the number sitting in the middle of a sorted set of numbers.

```javascript

// not the actual way to do it, just pointing out the idea
const median = (sortedArray) => sortedArray[array.length/2]

const x = [1,2,3,4,5]

median(x) // 3


```
